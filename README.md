# Bobcat300-TTN
Give your Bobcat 300 G285 Helium hotspot a second life by converting it into a fully functional The Things Network (TTN) LoRaWAN gateway.
Convert a Bobcat 300 Helium hotspot into a The Things Network (TTN) gateway.

## Features

- Backup existing Helium installation
- Install Semtech Basic Station
- Configure for The Things Network
- Easy installation using install_ttn.sh

## Supported hardware

| Model | Status |
|-------|--------|
| G285 | ✅ Tested |
| G290 | 🟡 Expected to work |
